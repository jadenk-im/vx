Protein is an app which uses the visual language of elemental pills to mimic chemical synthesis, allowing the user to deterministically distill any phrase or paragraph to its semantic average. It uses a structured hash function to map each word within a 32–dimensional vector space, calculating the centroids between words, and finding the nearest neighbor in a predefined vocabulary if the distance is within a threshold.
